### charAt()方法
   * 可返回指定位置的字符
   * 第一个字符位置为 0, 第二个字符位置为 1,以此类推
   > 
  ```
  var str = "HELLO WORLD";
  var n = str.charAt(2)
  ```
---------------
### concat()方法
  * 连接两个或更多字符串，并返回新的字符串。
  * 没有改变原有字符串，但是会返回连接两个或多个字符串新字符串。
> 
```
 var str1 = "Hello ";
 var str2 = "world!";
 var n = str1.concat(str2);
```
